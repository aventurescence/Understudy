# ![Understudy](Understudy/aSSETS/Understudy2.png) Understudy

![GitHub Downloads (all assets, all releases)](https://img.shields.io/github/downloads/aventurescence/understudy/total)


Your raid progression and currency tracker.

## Installation

1. Add `https://raw.githubusercontent.com/aventurescence/churinplugins/master/pluginmaster.json` to your Dalamud settings.
2. Search for "Understudy" in the plugin installer and install it.

## Description

Understudy helps you keep track of your weekly raid progression, tomestone caps, and gear sets across your characters.

## Features

- **Raid Tracking**: View your weekly clear status for current raid tiers.
- **Currency**: Monitor your limited tomestones and other currencies.
- **Gear Management**: Track gear sets for your jobs.

## Usage

- Use the slash command `/understudy` to toggle the main window.
- Access settings via the plugin installer or the config button in the main window.

## Building

1. Open `Understudy.sln` in your C# editor (Visual Studio 2022 or JetBrains Rider).
2. Build the solution (Debug or Release).
3. The resulting plugin is located in `Understudy/bin/x64/Debug/Understudy.dll`.
